# Kitty Config

Personal Kitty terminal configuration.

- `kitty.conf` contains fonts, padding, keybindings, shell, opacity, and theme includes.
- `current-theme.conf` defines the active color theme.
- `search.py` and `scroll_mark.py` provide the custom search kitten helpers.

Place this directory at `~/.config/kitty` and restart Kitty to apply changes.
